Let's start with an example:

curl -X PUT bashcache.com/key --data "value"
$ curl bashcache.com/key
value

Sounds easy, right? Lets try some advanced stuff:

Create a random hash key for a secret value:

$ curl -X PUT bashcache.com/$(openssl rand -hex 12) --data "secret value"
bashcache.com/0894abb1b622c49c9e7b3807

Try accessing it:

$ curl bashcache.com/0894abb1b622c49c9e7b3807

The resource can be globally locked: NOTE: This means, that the cache is locked until it's get purged! First kamm first surf! üèÑ

$ curl -X LOCK bashcache.com/0894abb1b622c49c9e7b3807

Any resource can be deleted, if it is not locked

curl -X DELETE bashcache.com/key

Another example for storing a txt file:

$ curl -X PUT bashcache.com/niklas-notes.txt --data "$(cat ~/Desktop/notes.txt)"
